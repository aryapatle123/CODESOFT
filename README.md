# Movie Genre Classification

## Project Overview

This project uses Machine Learning and Natural Language Processing (NLP) to predict the genre of a movie based on its plot summary.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* TF-IDF Vectorization
* Logistic Regression

## Features

* Text preprocessing
* TF-IDF feature extraction
* Movie genre prediction
* Trained model saved using Pickle

## Files

* MovieGenreClassification.ipynb
* movie_genre_model.pkl
* tfidf_vectorizer.pkl
